# eval-rubric

AI가 수행한 작업 결과물을 정량적으로 평가하기 위한 체크리스트와 점수표(과락 기준 포함)를 생성하고, 결과물이 과락 기준에 미달할 경우 구체적인 피드백을 제공하는 스킬.

## 사용법

```
/eval-rubric
```

실행 후 평가 기준을 만들 프롬프트를 붙여넣으면 된다. AI 결과물도 함께 제공하면 즉시 평가까지 수행한다.

## 동작 방식

### 1단계: 입력 받기

다음 두 가지를 입력받는다:

1. **평가 대상 프롬프트** (필수): AI에게 주어진 지시문
2. **AI 결과물** (선택): 평가할 AI의 답변 또는 산출물

AI 결과물이 없으면 평가 지표만 생성하고, 있으면 평가 지표 생성 후 즉시 평가를 수행한다.

### 2단계: 프롬프트 분석

프롬프트를 분석하여 다음을 파악한다:

- 작업 유형 (글쓰기, 코딩, 분석, 요약, 번역 등)
- 핵심 요구사항 목록
- 품질 판단에 필요한 기준
- 형식/구조 요건
- 맥락적 제약 조건

### 3단계: 평가 지표 생성

분석 결과를 바탕으로 아래 두 가지 산출물을 생성한다.

#### 체크리스트

프롬프트의 요구사항을 Yes/No로 확인할 수 있는 항목들.

```
## 체크리스트

### 필수 항목 (모두 충족해야 함)
- [ ] (항목)
- [ ] (항목)

### 권장 항목 (품질 향상 요소)
- [ ] (항목)
- [ ] (항목)
```

#### 점수표

각 평가 항목에 배점을 부여하고 과락 기준을 명시한다.

```
## 점수표

| 평가 항목 | 배점 | 과락 기준 | 설명 |
|-----------|------|-----------|------|
| (항목명)  | N점  | N점 미만  | (평가 기준 설명) |
| ...       | ...  | ...       | ... |

**총점**: 100점
**전체 과락 기준**: 70점 미만 (조정 가능)

### 과락 항목 안내
개별 항목 과락 시, 총점과 무관하게 전체 평가를 불합격으로 처리한다.
```

**점수 구성 원칙**
- 총점은 100점 기준
- 프롬프트의 핵심 요구사항일수록 높은 배점
- 형식·구조 요건은 별도 항목으로 분리
- 과락 기준은 각 항목 배점의 50% 이하로 설정 (단, 핵심 항목은 더 엄격하게)

### 4단계: 결과물 평가 (AI 결과물이 제공된 경우)

생성된 지표로 실제 결과물을 평가한다.

#### 체크리스트 평가

```
## 체크리스트 평가 결과

### 필수 항목
- [x] (충족된 항목)
- [ ] (미충족 항목) ← **미충족**

### 권장 항목
- [x] (충족된 항목)
- [ ] (미충족 항목)
```

#### 점수 집계

```
## 점수 집계

| 평가 항목 | 배점 | 획득 점수 | 과락 여부 |
|-----------|------|-----------|-----------|
| (항목명)  | N점  | N점       | 통과/과락 |
| ...       | ...  | ...       | ...       |

**총점**: N / 100점
**최종 판정**: 합격 / 불합격
```

#### 피드백 (과락 항목이 있는 경우)

```
## 피드백

### 과락 항목 상세

#### (과락 항목명) — N점 / N점 (기준: N점 이상)
- **문제점**: (무엇이 부족한지 구체적으로)
- **개선 방향**: (어떻게 수정하면 기준을 충족할 수 있는지)
- **예시**: (가능하면 개선 예시 제공)

#### ...
```

## 평가 항목 예시 (작업 유형별)

### 글쓰기 과제
| 항목 | 배점 |
|------|------|
| 주제 부합도 | 30점 |
| 논리적 구조 | 20점 |
| 문체/톤 적합성 | 20점 |
| 분량/형식 준수 | 15점 |
| 문법·맞춤법 | 15점 |

### 코드 작성 과제
| 항목 | 배점 |
|------|------|
| 기능 요구사항 충족 | 40점 |
| 코드 동작 정확성 | 25점 |
| 코드 가독성 | 20점 |
| 에러 처리 | 15점 |

### 분석/요약 과제
| 항목 | 배점 |
|------|------|
| 핵심 내용 포함 여부 | 35점 |
| 정확성 | 30점 |
| 간결성 | 20점 |
| 형식 준수 | 15점 |
