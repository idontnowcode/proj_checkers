# prompt-clarify

사용자가 작성한 AI 지시 프롬프트를 분석하여, 모호하거나 불충분한 부분을 찾아내고 역질문을 통해 프롬프트를 구체화·개선해주는 스킬.

## 사용법

```
/prompt-clarify
```

실행 후 분석할 프롬프트를 붙여넣으면 된다.

## 동작 방식

### 1단계: 프롬프트 입력 받기

사용자에게 개선하고 싶은 프롬프트를 입력해달라고 요청한다.

### 2단계: 명확성 분석 (최대 3라운드)

입력된 프롬프트를 아래 기준으로 분석한다:

**분석 항목**
- **목적/목표**: AI에게 기대하는 결과가 명확한가
- **맥락/배경**: 필요한 배경 정보가 충분히 제공되었는가
- **출력 형식**: 결과물의 형식, 길이, 구조가 명시되었는가
- **대상/범위**: 작업의 범위와 제약 조건이 분명한가
- **기준/조건**: 성공 기준이나 품질 조건이 포함되어 있는가
- **예시**: 예시나 참고 자료가 있으면 도움이 되는 상황인가

**라운드 진행 규칙**
- 모호하거나 누락된 항목에 대해 질문을 생성한다
- 질문 수는 실제 필요한 개수만큼만 (과도한 질문 금지)
- 사용자가 답변하면 재분석하여 여전히 불명확한 부분이 있으면 다음 라운드 진행
- 충분히 구체화되었다고 판단되면 라운드를 조기 종료
- 최대 3라운드 후 반드시 개선된 프롬프트를 출력

**각 라운드 출력 형식**
```
[라운드 N/3] 다음 사항을 확인하고 싶습니다:

1. (질문)
2. (질문)
...
```

### 3단계: 개선된 프롬프트 출력

모든 라운드가 끝나면 수집된 정보를 바탕으로 개선된 프롬프트를 출력한다.

**출력 형식**
```
## 개선된 프롬프트

(개선된 프롬프트 전문)

---
## 주요 개선 사항
- (원본 대비 달라진 점 목록)
```

## 예시 시나리오

**원본 프롬프트 (나쁜 예)**
> "마케팅 글 써줘"

**1라운드 질문 예시**
1. 어떤 제품/서비스에 대한 마케팅 글인가요?
2. 타깃 독자는 누구인가요? (연령대, 직군 등)
3. 글의 목적은 무엇인가요? (구매 유도, 브랜드 인지도, 이벤트 안내 등)
4. 글의 길이나 형식에 제약이 있나요? (SNS 포스트, 블로그, 광고 카피 등)

**개선된 프롬프트 예시**
> "30대 직장인을 타깃으로 한 SaaS 업무 자동화 툴의 SNS(인스타그램) 광고 카피를 작성해줘. 구매 유도가 목적이며, 150자 이내로 간결하게, 공감대를 자극하는 톤으로 작성해줘."
